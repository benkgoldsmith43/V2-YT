Privacy Policy

Last updated: June 2026

Overview

This is a personal dashboard application used solely by its owner. It is not a commercial product and does not collect, sell, or share any personal data with third parties.

Data Storage

All data entered into this dashboard (goals, health metrics, finances, study progress, etc.) is stored either:

  • Locally in your browser via localStorage, or
  • In a private Supabase database accessible only to the dashboard owner.

No data is shared with any third party, sold, or used for advertising.

WHOOP Integration

This app connects to the WHOOP API to display recovery, sleep, and heart rate data. When you authorize this connection:

  • WHOOP OAuth tokens are stored only in your browser's localStorage.
  • Health data fetched from WHOOP is displayed on-screen and optionally cached in your private Supabase database.
  • No WHOOP data is transmitted to any server other than the WHOOP API itself and your own Supabase instance.

API Keys

Any API keys you enter (e.g. Anthropic) are stored only in your browser's localStorage and are never sent to any server other than the respective API provider.

Cookies

This app does not use cookies for tracking or advertising. Session state is managed via sessionStorage and localStorage only.

Third-Party Services

  • WHOOP — health data API. Subject to WHOOP's privacy policy.
  • Supabase — database hosting. Subject to Supabase's privacy policy.
  • Anthropic — AI API. Subject to Anthropic's privacy policy.
  • Vercel — hosting. Subject to Vercel's privacy policy.

Contact

This dashboard is a private personal project. For questions, contact the owner directly.
